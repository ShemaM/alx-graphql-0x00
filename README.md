# Get a List of All Characters

## Objective

Create a GraphQL query to retrieve a paginated list of all characters.

## Instructions

Write a GraphQL query using the `characters(page: Int)` field to fetch the list of characters.
**Subfields selected:** id, name, status, and image.

## Files

- **characters-page-X.graphql**: Contains the query for page X.
- **characters-page-X-output.json**: Contains the JSON response for page X
